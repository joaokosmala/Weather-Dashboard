# 🌤️ Weather Dashboard

Um dashboard meteorológico moderno desenvolvido com HTML, CSS e JavaScript, capaz de consultar condições climáticas em tempo real para cidades do mundo inteiro.

Veja em: [Clique aqui](https://weatherdashboardtestsite.netlify.app/)

O projeto utiliza APIs externas para obter informações atualizadas sobre temperatura, velocidade do vento, sensação térmica e condições climáticas, exibindo os dados em uma interface intuitiva e responsiva.

---

## 🚀 Funcionalidades

### 🌍 Busca de Cidades

* Pesquisa de cidades em qualquer lugar do mundo.
* Consulta rápida e intuitiva.
* Dados atualizados em tempo real.

### 🌡️ Informações Climáticas

* Temperatura atual.
* Sensação térmica.
* Velocidade do vento.
* Condição climática atual.

### 🎨 Interface Dinâmica

* Ícones climáticos automáticos.
* Alteração dinâmica de fundo conforme o clima.
* Design moderno e responsivo.

### 📱 Responsividade

* Compatível com computadores.
* Compatível com tablets.
* Compatível com smartphones.

### ⚡ Desempenho

* Aplicação leve.
* Sem dependência de frameworks.
* Desenvolvida utilizando JavaScript puro.

---

## 🛠️ Tecnologias Utilizadas

| Tecnologia               | Finalidade                     |
| ------------------------ | ------------------------------ |
| HTML5                    | Estrutura da aplicação         |
| CSS3                     | Estilização e responsividade   |
| JavaScript (ES6+)        | Lógica da aplicação            |
| Fetch API                | Requisições HTTP               |
| Async/Await              | Operações assíncronas          |
| Open-Meteo API           | Dados meteorológicos           |
| Open-Meteo Geocoding API | Busca de cidades e coordenadas |
